# Size
Size は、指定したタイトルを持つウィンドウを、指定した位置とサイズに変更するプログラムです。

## 例
```
.\Size.exe PowerShell 460 20 1000 1000
```
→タイトルに “PowerShell” という文字列を含むウィンドウの、位置を (460, 20) に、サイズを 1000×1000 に変更します。
